# Introduction

The TIVR library allows to perform all the essential functions for ivr realization.

The available features are:

* Play audio message
* Collect digits
* Repeat collected digits
* Call recording
* Call routing
* Write logs
* Alarms generator
* Audio playback via tts
* Retrieve informations through ASR
* Calendar check
